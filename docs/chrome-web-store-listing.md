## Description

Typlx fixes grammar and spelling in any text field on any website — powered by the LLM of your choice.

Unlike cloud-dependent grammar checkers, Typlx sends text only to the API provider you configure. No intermediary servers, no data harvesting, no accounts required.

FEATURES

• Works everywhere — textareas, inputs, contenteditable, Gmail compose, SPAs.
• Bring your own model — OpenAI, Anthropic Claude, or any OpenAI-compatible API (Ollama, LM Studio, vLLM for fully local/offline use).
• Privacy first — your API key is encrypted at rest (AES-GCM). No telemetry, no analytics beacons, no phone-home.
• Per-site toggle — enable or disable Typlx on any site.
• Open source — MIT licensed. Audit every line of code on GitHub.

HOW IT WORKS

1. Click the Typlx icon and enter your LLM provider settings (API URL, model, token).
2. Focus any text field on any page.
3. Click the fix icon that appears in the bottom-right corner.
4. Your text is replaced with the corrected version.

PRIVACY

Typlx does not collect, transmit, or store any user data on third-party servers. Text is sent only to the LLM endpoint you configure. For maximum privacy, use a local model so nothing ever leaves your device. Full privacy policy: https://github.com/typlx/chrome-grammar-fix-extension/blob/main/PRIVACY_POLICY.md
